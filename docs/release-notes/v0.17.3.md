# v0.17.3

## ユーザーにとって良くなったこと

- ビューワーの操作バーが常に画面上部に固定表示されるようになりました（スクロールしても見失わない）
- 絞り込み（ユーザー・タグ・日付）を 1 つのモーダルにまとめました。タブで切り替えて条件を組み合わせられます
- 適用中の絞り込みをカプセル型チップで表示し、ワンクリックで内容確認・全解除が行えます
- 「一括タグ付け」ボタンから件数表示を取り除き、操作バーをすっきりさせました
- 並び順ボタンがギアアイコンと同じ行に収まるようレイアウトを修正しました

## 修正内容

- **sticky toolbar**: 操作バーを `position: sticky` で常時表示化。絞り込み・並び替え・設定をひとつのバーに集約
- **unified filter modal**: ユーザー・タグ・日付の 3 タブを 1 モーダルに統合。タブ間で状態を保持したまま切り替え可能
- **filter chip**: 適用中の絞り込みをカプセル型で表示。左側テキスト＋縦線区切り＋右側 × ボタンのデザイン
- **toolbar layout**: `flex-wrap: nowrap` でギアアイコンが 2 行目に折れ返らないよう修正
- **bulk tag button**: `archiveTotalCount` prop を削除しボタンをシンプル化

## 詳細

ビューワーの一覧操作 UI を大幅に整理したリリースです。

これまでは絞り込み条件（ユーザー・タグ・日付）がそれぞれ別のモーダルに分かれていましたが、1 つの統合フィルターモーダルに集約しました。タブを切り替えながら複数条件を同時に設定でき、適用中の絞り込みはカプセル型チップで一目で確認できます。

操作バー自体は `position: sticky` により常に画面上部に表示されるため、長い一覧をスクロールしながらでも絞り込みや並び替えがすぐに行えます。
