# v0.13.0

## 追加・変更
- viewer と settings に `ja` / `en` の言語設定を追加
- 一覧画面と設定画面の文言を言語設定に連動
- 保存時の built-in auto tag を `system_key` で保存する構造に変更
- `liked` / `image` / `video` の既知タグを表示時にローカライズ
- メディア付き投稿の保存時に画像・動画の built-in tag を自動付与
- アプリログの種類フィルタを複数選択方式に変更
- viewer と log filter の各コントロール配置を調整

## 補足
- 既知の built-in auto tag は表示文字列ではなく言語非依存キーで保持するようになりました。今後ロケールを増やす場合も、保存済みデータを書き換えずに表示言語だけ切り替えられます。
- 旧 legacy DB の自動削除処理は入れていません。必要なら拡張を再インストールして初期化できます。
