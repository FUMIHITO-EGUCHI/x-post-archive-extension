# v0.13.0

## ハイライト

- viewer と settings の日本語 / 英語表示に対応
- built-in auto tag をローカライズ対応の保存方式へ変更
- 既存データ移行を含め、タグ表示まわりの整合性を改善
- viewer とログフィルタの表示文言を見直し

## 変更内容

このリリースでは、viewer 全体の表示言語切り替えと、built-in auto tag のローカライズ対応を進めました。
`liked` / `image` / `video` のような既定タグは、表示文字列ではなく共通キーで保存する方式に寄せ、保存済みデータの整合性を保ちながら日本語と英語の切り替えに対応しています。

## 含まれる更新

- viewer / settings の `ja` / `en` 切り替えを追加
- built-in auto tag を `system_key` ベースで保持するよう更新
- 画像・動画・いいね由来タグの表示をローカライズ対応
- 既存データ向けの移行処理を追加
- viewer とログフィルタの表示文言を調整
