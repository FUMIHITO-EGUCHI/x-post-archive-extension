# v0.12.1

## ハイライト

- 閲覧画面の投稿一覧で、X のハンドル名に加えて表示名も確認できるようになりました。
- 投稿保存時に、その時点の表示名をスナップショットとして保存するようにしました。

## 変更内容

このリリースでは、保存済み投稿の見分けやすさを改善しています。
これまでは一覧上で主にハンドル名を頼りに投稿を識別していましたが、表示名の変更や似たアカウント名がある場合に判別しづらいことがありました。
`v0.12.1` では、投稿保存時に取得した表示名を保持し、閲覧画面の一覧でハンドル名とあわせて表示します。

## 含まれる更新

- X の投稿 DOM から `display_name` を抽出する処理を追加
- アーカイブの保存データに `display_name` を保持するよう更新
- viewer の投稿一覧 UI で `display_name` を表示

## 注意事項

- 既存の保存済み投稿には表示名が入っていないため、再保存するまでは従来どおりハンドル名のみ表示される場合があります。
