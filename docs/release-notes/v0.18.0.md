# v0.18.0

## ユーザーにとって良くなったこと

- フィルターモーダルのタグ一覧・ユーザー一覧が無限スクロールに対応しました。大量のタグ・ユーザーがある場合でも、スクロールするたびに追加で読み込むため初期表示が速くなります
- フィルターモーダルのスクロールバーが非表示になり、見た目がすっきりしました

## 修正内容

- タグ除外フィルターを追加（特定タグを含む投稿を除外できるように）

## 詳細

### 無限スクロール（フィルターモーダル）

タグ一覧・ユーザー一覧は表示件数が多い場合に初期ロードが重くなる問題がありました。`useInfiniteOptionList` フックを導入し、`IntersectionObserver` でセンチネル要素を監視して追加読み込みをトリガーする実装に変更しました。

投稿一覧（viewer-app）にも同様の無限スクロールを適用し、大量アーカイブでもスムーズにスクロールできるようにしました。

### モーダルスクロールバー非表示

`.viewer-modal` および `.viewer-tag-option-list` に `scrollbar-width: none` と `::-webkit-scrollbar { display: none }` を適用し、スクロールバーを視覚的に隠しました。スクロール自体は引き続き動作します。

### タグ除外フィルター

フィルターモーダルにタグ除外オプションを追加しました。指定したタグが付いた投稿を検索結果から除外できます。

## 含まれる変更

- `src/features/viewer/components/viewer-app.tsx`: 投稿一覧に無限スクロールを追加
- `src/features/viewer/components/unified-filter-modal.tsx`: タグ・ユーザー一覧に `useInfiniteOptionList` を適用。タグ除外フィルター UI を追加
- `src/entrypoints/viewer/style.css`: センチネル要素スタイル追加、モーダル・タグリストのスクロールバー非表示
