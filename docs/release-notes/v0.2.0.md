# v0.2.0

## ハイライト

- OPFS を使った画像アーカイブ閲覧に対応
- 投稿内の動画候補を抽出し、直接 MP4 を保存して再生できるよう改善
- 動画保存の取得経路を GraphQL ベースに整理

## 変更内容

このリリースでは、保存済みメディアの扱いを大きく前進させました。
画像をローカル保存して viewer から見返せる流れを整えつつ、動画についても候補抽出と直接保存の基盤を追加しています。

## 含まれる更新

- OPFS 保存画像の viewer 表示を追加
- 投稿から動画候補を抽出する処理を追加
- 直接 MP4 を保存して再生できる投稿に対応
- GraphQL レスポンスから動画候補を拾う処理を追加
