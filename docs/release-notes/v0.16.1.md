# v0.16.1

## 修正内容
- v0.16.0 で作成したバックアップ ZIP を復元すると `tag.system_key is invalid.` で失敗する不具合を修正
- `bookmarked` built-in tag を含むタグデータを復元時にも正しく受け付けるよう調整

## 詳細

v0.16.0 では bookmarks import と自動保存の追加にあわせて `bookmarked` built-in tag を保存できるようにしていましたが、復元側のバックアップ検証だけ許容値の更新が漏れていました。そのため、v0.16.0 で生成したバックアップ ZIP に `bookmarked` を持つタグが含まれていると、復元処理の入力検証で `tag.system_key is invalid.` が発生していました。

このリリースでは復元時の `tag.system_key` / `post_tag.system_key` 検証に `bookmarked` を追加し、v0.16.0 系で出力したバックアップをそのまま復元できるようにしています。

## 含まれる変更

- バックアップ復元時の built-in tag バリデーションに `bookmarked` を追加
- v0.16.1 へバージョン更新
