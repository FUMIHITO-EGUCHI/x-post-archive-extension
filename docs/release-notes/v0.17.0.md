# v0.17.0

## ユーザーにとって良くなったこと

- 保存済み投稿に複数のタグをまとめて付けられるようになりました（一括タグ付け）
- ビューアーで日付範囲フィルターが使えるようになりました
- ビューアーの並び順にランダムが追加されました
- フィルターリストがスクロールしながら段階的に読み込まれるようになり、大量データでも快適になりました
- 保存済み投稿を手動で再取得できるようになりました（Saved Post Refetch）
- likes 一括取得が重複続きのときに自動停止するようになりました
- 投稿カードがはみ出して表示される問題を修正しました

## 修正内容

- 一括タグ付け UI の追加と、バックグラウンドでのメタデータ再取得の改善
- 保存済み投稿の手動再取得機能の追加
- ビューアーの並び順にランダムを追加
- ビューアーの投稿カードのオーバーフローを修正
- likes 一括取得で重複が連続した場合の自動停止機能を追加
- ビューアーの日付範囲フィルターを追加
- フィルターリストへのインクリメンタルローディングを追加

## 詳細

v0.16.x でのバグ修正に続き、v0.17.0 はビューアーの機能拡充と一括操作の改善が中心です。

一括タグ付けにより、複数の投稿に対して同時にタグを設定できるようになりました。あわせてバックグラウンドでのメタデータ再取得が改善され、タグ反映の遅延が減っています。

ビューアー側では日付範囲フィルター・ランダムソート・インクリメンタルローディングが追加され、大量の保存投稿でも絞り込みや閲覧がしやすくなっています。

likes 一括取得では重複が連続した場合に自動停止するようになり、無駄なリクエストが走り続けるケースを防いでいます。
