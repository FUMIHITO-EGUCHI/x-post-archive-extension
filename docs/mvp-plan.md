# MVP Plan

## MVP Goal

最初のリリースでは、X の投稿を 1 件ずつ保存し、保存済み一覧を見て削除できる状態までを完成条件とする。

## Included

- X の投稿ごとの保存ボタン
- DOM からの投稿情報抽出
- `posts` ストアへの保存
- 保存済み判定
- viewer での一覧表示
- viewer での物理削除

## Excluded

- タグ
- 検索
- スレッド
- 画像 / 動画
- 反応数
- 論理削除
- 自動更新

## Done Criteria

1. X 上の投稿に `保存` ボタンが出る
2. 未保存投稿を押すと IndexedDB に保存される
3. 保存後に `保存済み` 表示へ変わる
4. viewer に保存済み投稿が `saved_at` 降順で出る
5. viewer から削除すると一覧から消える
